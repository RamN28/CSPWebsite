Arduino Project: RFID Lock

This innovative project is a sophisticated yet user-friendly access control system, developed using an Arduino Uno and an MFRC522 RFID reader.
It epitomizes the seamless integration of hardware and software in creating a smart security solution.

At its core, the system utilizes RFID technology for secure access control. When an RFID tag is presented to the MFRC522 reader, the system checks the tag's data.
Based on the programmed criteria, it makes a decision: either granting access with a prompt "Access Granted" or denying it with "Access Denied."
This immediate feedback ensures a quick and efficient user experience. We are currently trying to enhance this setup by adding an additional feature: a 2.42-inch OLED screen.
This screen will act as a dynamic display, bringing the results over from the Arduino's serial port. It would vividly show the access status in real-time,
adding an extra layer of interaction and user engagement.
